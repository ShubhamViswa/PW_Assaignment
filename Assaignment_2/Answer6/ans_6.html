1. HTML

    Primary Use Case Key Features/Benifits
    Building the structure of web pages
    • Simple and easy to Learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Case Key Features/Benifits
    Styling and layout of web pages
    • Allows for separation of content and presentation
    • Enables responsive design
    • Offers a wide range of styling option

  3. JavaScript

    Primary Use Case Key Features/Benifits
    Adding Interactivity and functionality to the web page
    • Can manipulate and modify web page content
    • Offer some functionality to the web page
    • Allows for server side scripting with node js
